# Third-party licenses

The playground loads third-party JavaScript and Python packages at runtime from
public package CDNs. These package artifacts are not vendored in this repository.

## Pyodide

Pyodide `314.0.0-alpha.2` is loaded from jsDelivr:

`https://cdn.jsdelivr.net/pyodide/v314.0.0a2/full/`

Pyodide is distributed under the Mozilla Public License 2.0.

## CodeMirror

CodeMirror modules are loaded from jsDelivr as ESM modules. The playground uses
these MIT-licensed packages:

- `codemirror` 6.0.2
- `@codemirror/lang-json` 6.0.2
- `@codemirror/lang-python` 6.2.1
- `@codemirror/lang-yaml` 6.1.3
- `@codemirror/lang-xml` 6.1.0
- `@lezer/highlight` 1.2.3

CodeMirror copyright: Copyright (C) 2018-2024 by Marijn Haverbeke and others.

## Python packages

The playground installs Python packages from PyPI with `micropip` inside
Pyodide. The primary packages are:

- `datamodel-code-generator[graphql]`
- `tdom`
- `micropip`

Their dependencies are resolved by PyPI metadata at runtime.
